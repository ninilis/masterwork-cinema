Цели дипломной работы:
В этой дипломной работе реализован сайт для бронирования билетов в кинотеатр и разработана информационную систему для администрирования залов, сеансов и предварительного бронирования билетов.

При разработке проекта «Идём в кино» использовался следующий стек технологий:
Frontend:
HTML5 — семантическая разметка страниц (index.html, hall.html, payment.html, ticket.html, admin.html, login.html).
CSS3 — стилизация интерфейса, включая кастомные свойства (CSS-переменные) для единообразного управления цветами, отступами, шрифтами, медиазапросы.
JavaScript (ES6+) — вся клиентская логика: динамическая отрисовка данных, обработка событий, drag-and-drop, взаимодействие с API, работа с локальным хранилищем (localStorage), модальные окна.
Bootstrap 5 — использование готовых компонентов (сетка, модальные окна, кнопки, формы) для ускорения вёрстки и обеспечения адаптивности.
SortableJS — библиотека для реализации drag-and-drop при создании сетки сеансов (перетаскивание фильмов на временные ленты залов).
QRCreator — генерация QR-кода на странице билета.

Backend / API:
REST API — взаимодействие с сервером через предоставленный эндпоинт https://shfe-diplom.neto-server.ru/. Использовались методы GET и POST для получения данных (фильмы, залы, сеансы), авторизации администратора, добавления/удаления сущностей, обновления схемы залов и цен, покупки билетов.
FormData — для отправки данных в multipart/form-data (например, при загрузке постера или передаче конфигурации зала).

Инструменты разработки:
Webstorm — среда разработки.
Git / GitHub — система контроля версий, хранение кода.
Браузерные DevTools — отладка, тестирование адаптивности, анализ сетевых запросов.
Chrome / Yandex / Microsoft Edge — для кросс-браузерной проверки.

Все технологии подобраны с учётом требований дипломного проекта: чистый JS без фреймворков, использование внешних библиотек только там, где это оправдано (drag-and-drop, QR-код), и полное взаимодействие с предоставленным API.
